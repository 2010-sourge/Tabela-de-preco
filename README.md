# 💳 Pricing Cards

Um componente de tabela de preços com três planos de assinatura — **Basic**, **Extended** e **Premium** — desenvolvido com HTML e CSS puro.

---

## 📸 Preview

> Três cards brancos lado a lado sobre fundo azul/roxo, cada um com cabeçalho colorido em forma de banner com o nome do plano e preço, lista de recursos com ícones de check (✔) e X, e botão "Purchase" na cor do plano.

---

## 🚀 Tecnologias

- **HTML5**
- **CSS3**

---

## 📁 Estrutura do Projeto

```
pricing-cards/
├── index.html
└── style.css
```

---

## ⚙️ Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/pricing-cards.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd pricing-cards
   ```

3. Abra o `index.html` diretamente no navegador ou via **Live Server** no VS Code:
   - Acesse em: `http://127.0.0.1:5500`

---

## 🎯 Funcionalidades

- Três cards de planos de preço com visual distinto por cor
- Cabeçalho em formato de banner/chevron com nome e preço do plano
- Lista de features com ícones ✔ (verde) e ✘ (vermelho)
- Botão **Purchase** estilizado na cor de cada plano
- Layout responsivo com cards lado a lado

## 🎨 Design

- Fundo da página: azul/roxo médio (`#5865f2` aprox.)
- Cards: fundo branco com bordas arredondadas e sombra suave
- **Basic**: verde (`#4CAF50`)
- **Extended**: laranja (`#E8692A` aprox.)
- **Premium**: ciano/teal (`#2ABFBF` aprox.)
- Cabeçalho do card com forma de seta/chevron apontando para baixo
